# Default Soundpack

This is the default soundpack for AccessiClock.

## Adding Sounds

Place audio files in this directory to add sounds to the default soundpack.

Supported formats:
- WAV (.wav)
- MP3 (.mp3)
- OGG (.ogg)

## Recommended Sounds

- `chime.wav` - Hourly chime sound
- `tick.wav` - Second tick sound (optional)
- `alarm.wav` - Alarm sound (optional)

## Creating Sounds

You can create your own sound files or find free sounds from:
- freesound.org
- zapsplat.com
- freesoundeffects.com

Make sure to respect licensing requirements for any sounds you use.
